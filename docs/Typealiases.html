  • DiscoveryResponseBlock

    The Discovery response received in methods which required a presenter view controller. It is the responsability of the developer to dismiss the web controller. - Parameter controller: The Mobile Connect controller which contains the web view. Should be dismissed by the developer. - Parameter operatorsData: The data received from the Discovery service. Can be nil in case an error occured. - Parameter error: The error which is sent in case the operatorsData is nil.
